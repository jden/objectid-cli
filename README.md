# objectid-cli
easily generate an objectid string from the command line

## usage
```console
$ objectid
54bea0a955793f0b00000001

```


## api
no flags or options

## installation

    $ npm install -g objectid-cli


## running the tests

From package root:

    $ npm install
    $ npm test
